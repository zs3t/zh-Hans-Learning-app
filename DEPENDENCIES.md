## 更新指南

### 安全更新步骤

1. **在测试环境中执行**：
   ```bash
   ./scripts/update-deps.sh
   ```

2. **验证功能**：
   ```bash
   npm run dev      # 开发服务器
   npm run build    # 生产构建
   npm run lint     # 代码检查
   ```

3. **如果测试通过，部署到生产环境**

### 回滚步骤

如果更新后出现问题：
```bash
# 恢复备份
cp backup-deps-*/package.json .
cp backup-deps-*/package-lock.json .
npm install
```
